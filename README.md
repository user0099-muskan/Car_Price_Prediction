
# Car Price Prediction Using Machine Learning

This project predicts the selling price of used cars using Machine Learning techniques in Python.

## Project Objective
To build a regression model that can estimate car prices based on features like year, fuel type, kilometers driven, and more.

## Dataset Information
The dataset includes:
- Car Name
- Year
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Training
5. Prediction

## Machine Learning Model
Linear Regression

## How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Open Jupyter Notebook or Google Colab

3. Run all cells step by step

4. Enter input values to predict car price

## Result
The model predicts car prices based on given features.
