#**Car Price Prediction**
This project predicts the selling price of cars using Machine Learning.
##**Dataset**
The dataset contains information like:
-Car Name
-Year
-Present Price
-Kms Driven
-Fuel Type
-Seller Type
-Transmission
-Owner
##**Technologies Used**
-Python
-Pandas
-Numpy
-Matplotlib
-Scikit-learn
##**Project Workflow**
1.Data Collection
2.Data Preprocessing
3.Exploratory Data Analysis
4.Model Training
5.Prediction
##**Machine Learning Model**
-Linear Regression
##**How to Run**
1.Install required libraries
2.Run the notebook
##**Result**
The model predicts car price based on input features.
